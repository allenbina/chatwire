What it is

chatwire is a small Python service that runs on your Mac, watches the iMessage database, and bridges messages to a configurable set of plugins. The default plugin is Telegram — you read and reply to your iMessages from any Telegram client (phone, desktop, web). A web UI is included for setup and live message inspection.

It's designed for the same people who run their own Jellyfin, Home Assistant, or k3s cluster. Headless install. Tailscale or Cloudflare Zero Trust as the intended access model. ntfy or Pushover baked in.

  • Plugin architecture — `pipx inject` to add. Auto-generated settings UI per plugin.
  • Headless & CLI-first — no GUI, no Electron, no 400 MB download.
  • No relay servers — messages never transit a third-party cloud unless you point a plugin at one.
  • Zero telemetry — your data stays on your hardware.

Install

# 1. install
pipx install chatwire

# 2. interactive setup — picks plugins, walks you through perms
chatwire setup

# 3. start the daemon
chatwire start

# or run as a launchd service:
chatwire install-service
