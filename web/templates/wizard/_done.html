Setup complete

Configuration saved to ~/.chatwire/config.json (chmod 600). The bridge agents need a restart to pick up new env vars.

Restart the agents

From a terminal on this Mac:

launchctl kickstart -k gui/$(id -u)/dev.chatwire.bridge
launchctl kickstart -k gui/$(id -u)/dev.chatwire.web

If you installed with a different label prefix, substitute that. chatwire doctor lists the currently-loaded labels.

What's next

  • Open the main UI — by the time you click, the web agent's restart will be visible (the page reloads itself).
  • Re-run any wizard step if you want to tweak settings.
  • Tail logs: chatwire logs -f
