Step 1 — macOS permissions

The bridge needs two macOS privacy grants. Both are tied to the python.org Python binary at /Library/Frameworks/Python.framework/Versions/3.14/Resources/Python.app (your install path may vary). Apple does not allow these to be granted by an installer — you'll click through System Settings yourself.
